# Github Dependency Parser

Github Dependency Parser is a node web service which outputs JavaScript file dependencies for a given repo.  

## Installation

```bash
#install dependencies
yarn

#fire up the app on your local server
yarn server  
```
## Usage
```node
var foo = {1: 'bar', 2: 'bat'}
```

## Testing
```bash
yarn test
```
